Basic API with PHP (1 day)

PHP is a server side scripting language, used to develop Web applications.

Related

Basic API with PHP (1 day)
Complex API with Symfony (2 days)

This training will show which are the foundations to create an good REST API using PHP.

Results

By the end of the course, participants will be able to:

  • Acquire the basic concepts on how APIs work
  • Build simple, yet efficient APIs
  • Start building more complex API avoiding the classical pitfals
  • Improve transparency in the infrastructure setup

Requirements:

  • Knowledge of PHP
  • PHP 7.2 and webserver installed
  • Editor (PHPStorm, Sublime or similar)

Audience:

  • Software developers

Program:

  • API basics (theory about REST and HTTP)
  • Project setup
  • Use of URLs
  • Use of Verbs
  • Headers
  • Content handling
  • Error handling
  • Authentication

Format

The training will be provided as presentation, discussion, exercises and mainly hands-on practice.

Pricing

Starting at: 850 €

Training location: Berlin or client offices (the price does not include commute expenses)
Remote training sessions are 15% discounted.
The training is for a maximum of 4 people
