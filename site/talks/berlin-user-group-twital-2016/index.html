Twital, Feel power of Twig using an AngularJS-like syntax

Introduction to Twital, a PHP template engine

Symfony User Group - Berlin, Germany
2016-02

Twital is a PHP template engine based on Twig and adds some shortcuts and makes Twig's syntax more suitable for HTML based (XML, HTML5, XHTML, SGML) templates. Twital takes inspiration from PHPTal, TAL and AngularJS (just for some aspects), mixing their language syntaxes with the powerful Twig templating engine system.

More info are available on the project page https://github.com/goetas/twital
